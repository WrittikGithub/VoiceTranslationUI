# VoiceTranslationUI — Langlex AI Website

A modern Next.js website for voice translation services (VoiceTranslation full website), built with TypeScript and Tailwind CSS.

## Features

- Clean, modern design matching the Langlex AI brand
- Responsive layout for all devices
- Hero section with voice waveform visualization
- User metrics and statistics
- Pricing plans
- Testimonials section
- AI integrations showcase
- Footer with comprehensive links

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```
